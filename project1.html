Contact Manager

A recent project I completed was a web-based contact manager for my Processes of Object-Oriented Software class. Working with a team of developers, we built the application from the ground up, setting up a remote server, designing the frontend interface, managing the backend database, and creating API endpoints to connect the system end-to-end.

In this project, I served as both Project Manager and Database Developer. Using a DigitalOcean droplet, I configured the remote server, linked a custom domain, and implemented a MySQL database to handle user authentication and contact storage. Each account stored login credentials (name, username, and password) and supported an unlimited number of contacts, each with fields for name, email, and phone number. Users could add, edit, or delete their contacts at any time.

As Project Manager, I ensured clear communication, task organization, and steady progress throughout development. We coordinated via Discord, used GitHub for version control, and held weekly team meetings to align goals and track milestones.
